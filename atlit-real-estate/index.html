הפעל שרת סנכרון

כדי לסנכרן, יש להריץ את שרת הסנכרון על המחשב שלך (בישראל). פתח טרמינל בתיקיית הריפו והרץ:

pip install requests python atlit-real-estate/sync_server.py

השרת ירוץ ברקע — השאר את הטרמינל פתוח. לאחר מכן לחץ שוב על סנכרון.
